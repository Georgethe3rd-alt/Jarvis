# Executive Reporting Skill

Generate structured executive reports with proper formatting and strategic analysis.

## Report Types

### Financial Analysis
- Revenue projections
- Cost-base analysis
- Margin modeling
- Scenario planning

### Strategic Memos
- Regulatory positioning
- Commercial leverage assessment
- Risk analysis
- Next steps recommendation

### Board Presentations
- Structured narratives
- Data visualization prep
- Executive summaries
- Strategic recommendations

## Output Formats

1. **Executive Memo**
   ```
   TO: [Recipient]
   FROM: George the 3rd
   DATE: [Date]
   RE: [Subject]

   EXECUTIVE SUMMARY
   [Key points]

   ANALYSIS
   [Detailed findings]

   RECOMMENDATIONS
   [Strategic next steps]
   ```

2. **Financial Model**
   - Gross vs Net breakdowns
   - Margin analysis
   - Scale impact projections
   - Recurring revenue modeling

## Usage

When executive reporting is needed:
1. Identify report type required
2. Gather relevant data
3. Structure according to templates
4. Focus on strategic implications
5. Provide clear recommendations

Remember: Wayne operates at board/ministerial level. Keep analysis at executive intelligence level.
