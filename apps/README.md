# OpenClaw Applications

This directory contains all scripts, integrations, and custom skills developed for OpenClaw operations.

## Structure

```
apps/
├── scripts/       — Automation scripts and utilities
├── integrations/  — API integrations (AgentMail, Twilio, ElevenLabs, etc.)
├── agents/        — Sub-agent definitions and delegation examples
└── skills/        — Custom OpenClaw skills
```

## Scripts
- Webhook handlers
- Data processing utilities
- Automation workflows

## Integrations
- AgentMail email handling
- Twilio voice/SMS tools
- ElevenLabs TTS utilities
- GitHub automation

## Skills
- Custom OpenClaw skills for specific tasks
- Reusable components
- Domain-specific capabilities
